Base de dados para OPL

Bem vind-!

Esta é uma base de dados com descrições e artes de capa para jogos de PS2. Você pode utilizá-la com o OPL (Open PS2 Loader) para baixar tanto as capas quanto os arquivos de configuração (.cfg) para seus jogos.

Atenção

As artes foram obtidas do banco de dados de "danielb" no Internet Archive, e as descrições vêm do repositório de "Tom-Bruise", que está realizando um ótimo trabalho na criação desta grande base de dados.

As descrições dos jogos estão, originalmente, em inglês e francês. Com a ajuda do Google Gemini, estou traduzindo o conteúdo para o português. No entanto, para garantir que tudo esteja coerente, este processo pode levar algum tempo.

Como Utilizar

Utilize o campo de pesquisa a baixo para encontrar o jogo que deseja, você pode utilizar o código do jogo, o nome ou palavras chaves.

Para API
