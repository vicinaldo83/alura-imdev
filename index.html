Base de dados para OPL

Bem vind-!

Essa é uma base de dados de descrições e artes de capa para jogos de PS2. Você pode utiliza-la junto com o OPL para baixar tanto a capa quanto o arquivo .cfg de seu jogo.

Atenção

As artes são do banco de dados do danielb no Internt Archive e as descrições são do repositório do Tom-Bruise onde estão fazendo um ótimo trabalhando criando uma grande base de dados

As descrições dos jogos atualmente estão disponíveis apenas em inglês e francês. Com ajuda do Google Gemini, estou tentando traduzir para português o máximo que eu posso, mas para garantir que tudo esteja coerente, este processo pode demorar um pouco.

Como Utilizar

Utilize o campo de pesquisa a baixo para encontrar o jogo que deseja, você pode utilizar o código do jogo, o nome ou palavras chaves.

Para API

Pretendo também transformar esse site em uma API para facilitar o download desses arquivos e implementar isso no PyOPLM. Mas ainda é um trabalho em progresso
